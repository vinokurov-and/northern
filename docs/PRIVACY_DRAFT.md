## 2. Правовые основания

Обработка персональных данных осуществляется в соответствии с
Федеральным законом №152-ФЗ «О персональных данных». Правовые основания
обработки:

- **Согласие субъекта** — для действий, требующих явного выбора
  пользователя (например, клик по кнопке «Сделать прогноз», переход в
  приложение GameChallenge).
- **Законный интерес Оператора** — для обеспечения работы Сайта, защиты
  от злоупотреблений, базовой аналитики посещаемости.
- **Исполнение договора** — для функций приложения GameChallenge, к
  которым Сайт предоставляет доступ.

## 3. Какие данные мы собираем

### 3.1. Автоматически при посещении страниц

- **Анонимный идентификатор устройства** — случайно сгенерированное
  значение (UUID v4), сохраняемое в локальном хранилище вашего браузера
  (localStorage) по ключу `northern-anon-id`. Этот идентификатор **не
  связан** с вашим именем, email, номером телефона или иными прямыми
  идентификаторами. Мы используем его, чтобы:
  - не учитывать один и тот же визит в статистике повторно;
  - измерять уникальных посетителей страниц.
- **Хешированный IP-адрес** — ваш IP-адрес не сохраняется в исходном
  виде. Мы вычисляем его криптографический хеш (SHA-256, первые 16
  символов) и сохраняем только хеш. По хешу невозможно восстановить
  исходный IP. Используется для ограничения частоты запросов (защита от
  злоупотреблений).
- **User-Agent** строка браузера (модель браузера, операционная
  система).
- **Referer** — адрес страницы, с которой вы перешли на Сайт (если
  браузер его передал).
- **Путь посещённой страницы** на нашем сайте (например, `/players`).

Эти данные сохраняются на срок до **12 месяцев** с момента последнего
посещения и используются исключительно для внутренней аналитики
посещаемости Сайта.

### 3.2. При взаимодействии с кнопками и ссылками

Мы фиксируем факт нажатия на отдельные кнопки Сайта (например, кнопку
перехода к приложению прогнозов GameChallenge), сохраняя идентификатор
кнопки, анонимный идентификатор устройства и хешированный IP. Цель —
измерение эффективности разных элементов интерфейса.

### 3.3. При переходе по рекламным ссылкам

На Сайте размещаются рекламные баннеры партнёрских сетей. При клике по
такой ссылке вы переходите на сайт рекламодателя или партнёра, где
действует его собственная политика конфиденциальности. Мы не передаём
партнёрам ваши персональные данные — передаются только технические
параметры рекламной кампании.

### 3.4. Что мы НЕ собираем

- Имя, фамилию, отчество.
- Адрес электронной почты, номер телефона.
- Данные платёжных карт (Сайт не принимает платежи).
- Геолокацию (точные координаты).
- Данные с камеры, микрофона.
- Cookies сторонних аналитических систем (Google Analytics, Яндекс
  Метрика и подобных на Сайте нет).

## 4. Где и как хранятся данные

Данные обрабатываются и хранятся на серверах, используемых Оператором
для работы Сайта. Передача данных осуществляется по защищённому каналу
(HTTPS, TLS 1.2+). Доступ к серверам ограничен Оператором.

## 5. Трансграничная передача персональных данных

Оператор вправе осуществлять трансграничную передачу персональных
данных на территорию иностранных государств, обеспечивающих адекватную
защиту прав субъектов персональных данных в соответствии с перечнем
Роскомнадзора. Цель такой передачи ограничена обеспечением
технического функционирования Сайта.

## 6. Передача данных третьим лицам

Оператор **не передаёт** собранные данные третьим лицам, за исключением
случаев:

- Когда это требуется по закону (запрос уполномоченного государственного
  органа).
- Когда этого требует работа рекламных партнёрских сетей в рамках
  перехода по рекламным ссылкам (см. §3.3) — в этом случае передаются
  только технические параметры кампании, без персональных данных.

## 7. Ваши права

В соответствии с 152-ФЗ вы имеете право:

- Получить информацию о том, какие данные о вас обрабатываются.
- Потребовать удаления своих данных.
- Потребовать уточнения или блокировки данных.
- Отозвать согласие на обработку данных.

Для реализации любого из этих прав напишите на адрес, указанный в §1.
Мы ответим в течение **30 календарных дней**.

Чтобы быстро удалить анонимный идентификатор устройства, вы также
можете очистить данные сайта через настройки браузера (раздел «Куки и
данные сайтов» или аналогичный). После этого при следующем посещении
будет сгенерирован новый анонимный идентификатор.

## 8. Связанные сервисы

`fc-sever.ru/app`). Приложение GameChallenge имеет собственную Политику
